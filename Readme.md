# AI Powered Healthcare & Mental Fitness Tracker

This project is an end-to-end AI-powered healthcare system that combines:
- Mental health prediction using Machine Learning (SVM, Random Forest, NLP)
- Stress detection using questionnaire-based SVM classification
- AI chatbot using Deep Learning (PyTorch + NLP)
- Web application using Flask

## Features
- Depression and suicide risk detection using TF-IDF + ML models
- Stress prediction using 10-question SVM model
- AI chatbot trained on mental health conversations
- User login & signup system
- Real-time web interface

## Tech Stack
- Python, Flask
- scikit-learn, Pandas, NumPy
- PyTorch (chatbot)
- NLP (NLTK, TF-IDF)
- Excel & CSV based data storage

## How to Run
pip install -r requirements.txt
python app.py
